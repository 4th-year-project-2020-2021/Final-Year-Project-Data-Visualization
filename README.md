## How to run the project
Launch with:
<br>
`git clone https://github.com/4th-year-project-2020-2021/Final-Year-Project-Data-Visualization.git`
<br>
`cd Final-Year-Project-Data-Visualization`
<br>
`npm install`
<br>
`npm start`
